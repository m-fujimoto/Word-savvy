Word-savvy manual

If you want to improve your English, Word-savvy might be fab. Consult good dictionary, if you have any question.

What is Word-savvy

Word-savvy is tiny database to review words.

How to use

  • Click Add button to insert a word into Word-savvy.db (SQlite).
  • Click Reset button to set default value.
  • Click Close button to exit this programme.
  • Double click WORD on data-table to see more details.
  • After displaying more details, if you want to edit / delete a data,click Edit / Delete button. You must tick (check) Edit / Delete box.

How to use STAR

The STAR is something like reminder. If you set something,five white stars are also Okay, it could be used by Review later. You must tick (check) Edit / Delete the box. Click STAR on data-table which you want to modify it. Then a white star is replaced a black star. If you set five black star, it would be back five white stars. This feature would be useful when you use Oxford menu.

How to search data

The WORD is required to search data.
  • Click Search KEYWORD by WORD button to search a word.
  • Click Search KEYWORD by PRONUNCIATION button to search a word.
  • Click Search KEYWORD by MEMO button to search a word.

Review button

If you want to search how many stars you set, use Review button. You can set a STAR of combobox as KEYWORD, then click Review button.

English-English Dictionary Menu

If you click on English-English menubar, you could select a lot of online Engilish-English Dectionaries. First, you might think that it is very difficult for me to use it. See carefully, you can find a lot of example to use it.

English-Japanese Dictionary Menu

If you click on English-Japanese menubar, you could select a lot of online Japanse-English Dctionaries. Use these dictionaris, if you are not familiar with English-English Dictionary.

Japanese Menu

If you want to know how Japanese means, use Japanese-language dictionary. If you want to know how you write Kanji,use mojinavi.

Oxford Menu

If you click on Oxford menubar, you could select a lot of word lists. To lean Oxford Learner's Word Lists, your vocabulary would increase effectively.So you could read even BBC Website in future.

What is the CEFR?

  • A1 and A2 indicate elementary and pre-intermediate levels of ability.
  • B1and B2 indicate lower- and upper-intermediate levels.
  • C1 indicates advanced level.
  • C2 indicates complete proficiency in the language.
